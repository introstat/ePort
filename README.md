Report Generating System for Intro Stat Education
============

[![Build Status](https://travis-ci.org/introstat/ePort.svg?branch=master)](https://travis-ci.org/introstat/ePort)

Input
------------

- Data files
- Answer keys
- Learning Outcomes

Output
------------

- reports for instructor
  - single homework
  - over the semester
  
- reports for coordinator
  - single homework
  - over the semester
